## Usage
To use these classes, fork repository, just copy and paste code to your solution/project or install nuget package. :smile:

<!-- INSTALLATION -->
## Installation

1. Install Nuget Package on your project
```sh
Install-Package UsefulHelpers -Version 1.0.0
```
